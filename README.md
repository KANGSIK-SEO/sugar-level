# 🩺 당뇨병 위험도 평가 시스템

당뇨병 예측 및 식단 분석 서비스입니다. **Mayo Clinic의 의료 기준**을 기반으로 개인의 당뇨병 위험도를 평가하고 맞춤형 권장사항을 제공합니다.

## ✨ 주요 기능

### 1. **당뇨병 위험도 평가**
- 공복 혈당(FBS) 분석
- 신체질량지수(BMI) 계산
- 식단 점수 평가
- 종합 위험도 산출

### 2. **건강 지표 분석**
- BMI 분류 (저체중, 정상, 과체중, 비만)
- 혈당 범주 (정상, 전당뇨, 당뇨병)
- 식단 영양소 분석 (탄수화물, 당분, 섬유질, 단백질)
- 생활 습관 평가 (당분 음료, 알코올)

### 3. **수명 예측**
- Mayo Clinic 기준 예상 수명 계산
- 당뇨병 위험도에 따른 수명 감소 추정

### 4. **맞춤형 권장사항**
- 혈당 관리 조언
- 체중 관리 가이드
- 식단 개선 방안
- 정기 검진 일정

### 5. **건강 관리 팁**
- 식단 관리 (5가지 팁)
- 운동 계획 (5가지 팁)
- 정기 검진 모니터링 (5가지 팁)

## 🔒 민감정보 보호

- **공복 혈당(FBS)**: 입력박스로 제공 - 사용자가 직접 입력
- **기타 의료 정보**: 클라이언트 사이드 처리, 서버에 최소 저장

## 🚀 설치 및 실행

### 필수 요구사항
- Node.js 14.0 이상
- npm 또는 yarn

### 설치 방법

```bash
# 레포지토리 클론
git clone https://github.com/KANGSIK-SEO/sugar-level.git
cd sugar-level

# 패키지 설치
npm install

# 개발 서버 실행
npm run dev

# 또는 프로덕션 실행
npm start
```

### 접속
```
브라우저에서 http://localhost:3000 접속
```

## 📊 위험도 분류

| 위험도 | 범위 | 설명 |
|--------|------|------|
| 낮음 | 0-30% | 건강한 상태 유지 |
| 중간 | 30-60% | 전당뇨 단계, 생활 습관 개선 권장 |
| 높음 | 60-80% | 전문의 상담 권장 |
| 매우 높음 | 80-100% | 즉시 전문의 상담 필수 |

## 📈 Mayo Clinic 기준

### 공복 혈당 (Fasting Blood Sugar)
- **정상**: < 100 mg/dL
- **전당뇨**: 100-125 mg/dL
- **당뇨병**: ≥ 126 mg/dL

### BMI 분류
- **저체중**: < 18.5
- **정상 체중**: 18.5-24.9
- **과체중**: 25-29.9
- **비만 1단계**: 30-34.9
- **비만 2단계**: ≥ 35
